# E-Commerce Sales Performance Dashboard with Cohort Analysis

## Project Overview
This project analyzes e-commerce sales performance and customer retention using Power BI dashboards and cohort analysis.

## Tools Used
- Power BI
- Excel
- DAX

## Key Features
- Sales KPI Dashboard
- Country-wise Sales Analysis
- Top Customers Analysis
- Cohort Heatmap

## Key Insights
- United Kingdom generated highest sales
- Sales showed seasonal trends
- Customer retention decreased over time
